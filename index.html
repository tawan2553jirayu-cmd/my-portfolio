Portfolio ด้านเทคโนโลยีสารสนเทศ

นาย จิรายุ วัฒนากรแก้ว | โรงเรียนบุญวาทย์วิทยาลัย

ประวัติ แรงบันดาลใจ ทักษะ ผลงาน เป้าหมาย

1. ประวัติส่วนตัวและการศึกษา

กำลังศึกษาอยู่ที่โรงเรียนบุญวาทย์วิทยาลัย สายการเรียนวิทยาศาสตร์-คณิตศาสตร์ แผนการเรียนคอมพิวเตอร์ มีพื้นฐานด้านไอทีและการเขียนโปรแกรมเบื้องต้น

2. ความสนใจและแรงบันดาลใจด้านไอที

มีความสนใจด้านเทคโนโลยีสารสนเทศจากการใช้งานคอมพิวเตอร์และโปรแกรมพื้นฐาน ชื่นชอบการคิดวิเคราะห์และแก้ปัญหาอย่างเป็นระบบ เชื่อว่าด้านไอทีสามารถนำไปพัฒนาสังคมและคุณภาพชีวิตได้

3. ทักษะและความรู้ด้านไอที

ด้านทักษะ รายละเอียด เครื่องมือ / ภาษา ระดับ
การเขียนโปรแกรม เขียนโปรแกรมพื้นฐานและแก้ปัญหา Python พื้นฐาน
พัฒนาเว็บไซต์ โครงสร้างเว็บไซต์ HTML, CSS พื้นฐาน
พัฒนาเว็บไซต์ เพิ่มการโต้ตอบ JavaScript กำลังพัฒนา
เครื่องมือพัฒนา จัดการโค้ด VS Code ใช้งานได้
การออกแบบ ออกแบบสื่อ Canva ใช้งานได้

4. ผลงานหรือโครงงาน

เคยสร้างเว็บไซต์พื้นฐานและทดลองเขียนโปรแกรมเพื่อฝึกทักษะด้านตรรกะและการแก้ปัญหา ได้เรียนรู้กระบวนการคิดอย่างเป็นขั้นตอน และการพัฒนาตนเองจากการลงมือทำจริง

5. เป้าหมายในอนาคต

มีเป้าหมายศึกษาต่อด้านเทคโนโลยีสารสนเทศที่มหาวิทยาลัยเชียงใหม่ เพื่อนำความรู้ไปต่อยอดสร้างสรรค์ผลงาน พัฒนาตนเอง และสร้างประโยชน์ให้กับสังคมในอนาคต
